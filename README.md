# 🇺🇸 American Tech English Handbook

面向中国程序员和技术管理者的地道美式英语学习手册。

2100+ 条覆盖日常交流、职场沟通、软件工程、AI、管理和投资的地道美式英语表达。

## 内容结构

| Book | 章节 | 表达数 |
|------|------|--------|
| Book 1: Core Spoken English | Ch1-3 | 750 |
| Book 2: American Workplace English | Ch4-7 | 300 |
| Book 3: Software Engineer English | Ch8-13 | 450 |
| Book 4: Tech Lead English | Ch14-16 | 225 |
| Book 5: AI & Startup English | Ch17-19 | 225 |
| Book 6: Investment English | Ch20-22 | 225 |
| **Total** | **22 chapters** | **2100+** |

## 技术栈

- **Frontend:** React 19 + TypeScript + Vite
- **UI:** TailwindCSS 4
- **State:** Zustand
- **Search:** Fuse.js
- **Data:** JSON (按章节拆分)
- **Deployment:** GitHub Pages

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `dist/` 目录。

## 内容生成

所有内容通过多 agent 工作流自动生成，经过严格的质量控制：
- 基于 COCA 语料库的高频表达选择
- 多视角交叉验证 (频率/自然度/实用性)
- 纯正美式英语，无教材腔
